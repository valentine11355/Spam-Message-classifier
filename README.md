# Spam-Message-classifier
# 📩 Spam Message Classifier

## Overview
This project classifies SMS messages as Spam or Not Spam using Machine Learning.

## Model
- TF-IDF Vectorizer
- Linear SVM (High Accuracy ~98–99%)

## Tech Stack
- Python
- pandas
- scikit-learn
- Streamlit

## Features
- Real-time spam detection
- Clean UI with Streamlit
- High accuracy
